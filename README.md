# **Flash Cards - JavaScript Quiz App**
A simple, interactive flash card application for learning JavaScript concepts. Built with vanilla HTML, CSS, and JavaScript.
Features

**20 pre-loaded JavaScript questions and answers**
Interactive card flipping animation
Progress tracking with visual progress bar
Keyboard navigation (Arrow keys and Spacebar)
Shuffle and reset functionality
Fully responsive design

**How to Run**

Download the flashcards.html file
Open it in any modern web browser
Start learning!

No installation or dependencies required.
Usage


Click the card to flip and see the answer
Previous/Next buttons to navigate between cards
Keyboard shortcuts: ← → arrows to navigate, Spacebar to flip
Shuffle to randomize card order
Reset to return to original order


Topics Covered
Variables, closures, promises, async/await, event loop, hoisting, array methods, destructuring, spread operator, template literals, and more.
Customization


To add your own questions, edit the flashcardsData array in the JavaScript section:


javascriptconst flashcardsData = [
    {
        id: 1,
        question: "Your question?",
        answer: "Your answer."
    }
];
Technologies

HTML5
CSS3 (with animations and 3D transforms)
Vanilla JavaScript (ES6+) #
